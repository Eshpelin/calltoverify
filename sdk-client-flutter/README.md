# sdk-client-flutter

Mobile client SDK for Flutter. The mobile counterpart of [`widget-web`](../widget-web): renders
verification instructions and tracks status inside a Flutter app.

> **Status: planned (Phase 4).** Placeholder directory.

## Scope

- Render channel instructions with native share/SMS/dial intents.
- Subscribe to status updates and surface `verified`.
- Pluggable transport so the app talks to its own backend, not the Coordinator directly.

## Planned stack

Dart / Flutter, published to pub.dev.
